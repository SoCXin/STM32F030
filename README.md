## [描述](https://github.com/sochub/STM32F030) 

[ST](https://github.com/sochub/ST) 意法半导体基于ARM Cortex®-M0的STM32 F0系列单片机实现了32位性能，同时传承了STM32系列的重要特性，特别适合成本敏感型应用。
